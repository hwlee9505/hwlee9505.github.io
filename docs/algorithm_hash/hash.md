# 해시

해시 알고리즘 문제 해결 전략  

해시는 Key-value쌍으로 데이터를 저장하는 자료구조 입니다.

---

### 해시 테이블

해시 테이블은(hash table)은 효울적인 탐색을 위한 자료구조로서 키(key)를 값(value)에 대응시킨다.  
해시테이블은 구현하는 방법은 여러가지가 있다.  

```scss
 hash table = LinkedList + hashCode()
```
